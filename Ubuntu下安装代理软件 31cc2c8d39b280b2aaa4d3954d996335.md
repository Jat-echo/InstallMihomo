# Ubuntu下安装代理软件

### 1. 下载 Clash 核心

首先,下载适合你系统架构的 Clash 核心文件。对于 AMD64 架构:

```bash
mkdir -p /opt/clash && cd /opt/clash

# 使用 GitHub 加速链接下载
wget -O mihomo.gz https://gh-proxy.com/https://github.com/MetaCubeX/Mihomo/releases/download/v1.19.2/mihomo-linux-amd64-v1.19.2.gz
gunzip mihomo.gz
chmod +x mihomo

wget -O geoip.metadb https://gh-proxy.com/https://github.com/MetaCubeX/meta-rules-dat/releases/download/latest/geoip.metadb
```

### 2. 添加配置文件

将你的 Clash 配置文件(config.yaml)上传到 `/opt/clash/` 目录:

[ClashMeta-UAH-20260602.yaml](ClashMeta-UAH-20260602.yaml)

```bash
# 请将下面的 URL 替换为你的真实订阅链接
curl -o /opt/clash/config.yaml "你的订阅链接URL"
```

### 3. 创建 systemd 服务

创建一个 systemd 服务文件以便 Clash 开机自启:

```bash
cat <<EOF > /etc/systemd/system/clash.service
[Unit]
Description=Mihomo Clash Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/clash
ExecStart=/opt/clash/mihomo -d /opt/clash
Restart=on-failure
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF
```

### 4. 启动和启用服务

重载 systemd 配置,启动并设置 Clash 开机自启:

```bash
systemctl daemon-reload
systemctl start clash
systemctl enable clash
```

### 5. 检查服务状态

验证 Clash 服务是否正常运行:

```bash
systemctl status clash
```

### 6. 配置系统代理(可选)

如果需要让系统流量通过 Clash,可以设置环境变量:

```bash
echo 'export https_proxy=http://127.0.0.1:7890' >> ~/.bashrc
echo 'export http_proxy=http://127.0.0.1:7890' >> ~/.bashrc
source ~/.bashrc
```

### 7. 测试连接

使用 curl 测试代理是否工作:

```bash
curl -x http://127.0.0.1:7890 https://www.google.com
```

如果返回 Google 页面内容,说明代理配置成功。

 

### 8. 远程访问 Clash 控制面板

编辑 Clash 配置文件,确保包含以下设置:

```yaml
# 在 /opt/clash/config.yaml 中添加或修改
external-controller: 0.0.0.0:9090
external-ui: ui
secret: '你的密码'  # 建议设置一个强密码
```

在配置文件中指定 UI 目录:

```bash
cd /opt/clash

# 1. 下载面板压缩包 (使用加速代理)
wget -O ui.zip https://gh-proxy.com/https://github.com/MetaCubeX/metacubexd/archive/refs/heads/gh-pages.zip

# 2. 解压
apt install unzip -y
unzip ui.zip
mv metacubexd-gh-pages ui

```

重启服务

```bash
systemctl restart clash
```

在浏览器中访问:

```
http://服务器IP:9090/ui
```

输入你设置的 `secret` 密码即可登录管理面板。
